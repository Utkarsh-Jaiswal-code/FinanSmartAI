# FinanSmartAI — AI Powered Personal Finance & Expense Tracking

FinanSmartAI is a full-stack AI-powered personal finance management system that helps users track expenses, plan budgets, generate insights, and get personalized financial recommendations using AI.

---

## Tech Stack

| Layer | Technologies Used |
|-------|------------------|
| Frontend | Next.js 15 (App Router), React.js, Tailwind CSS, ShadCN UI |
| Backend | Node.js, Drizzle ORM, REST APIs |
| Authentication | Clerk (Sign in / Sign up / Session Management) |
| Database | Neon (PostgreSQL) + Drizzle ORM |
| Charts | Recharts |
| AI / ML | OpenAI API (AI Recommendations Engine) |
| Deployment | Vercel (Frontend + Serverless Functions) |

## Key Features

1. AI-generated spending insights  
2. Track expenses by category  
3. Manage income and budgets  
4. Secure authentication using Clerk  
5. Visual analytics via charts  
6. PostgreSQL database using Drizzle ORM  
7. Fully responsive & modern UI
